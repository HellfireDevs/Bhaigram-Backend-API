# Contributing to Bhaigram Backend

First off, thank you for considering contributing to the Bhaigram Backend API! It's people like you that make open-source software such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check if there's already an [issue](https://github.com/yourusername/bhaigram-backend/issues) open for it. If not, feel free to open one!

## Setting up for Local Development

1. Fork the repo and create your branch from `main`.
2. Ensure you have Python 3.10+ installed.
3. Install dependencies: `pip install -r requirements.txt`
4. Create a `.env` file and set up your `IG_SESSION_ID` and `MONGO_URI`.
5. Run the local server: `uvicorn main:app --reload`
6. Make your changes and test them locally.

## Pull Requests

1. Keep PRs small and focused on a single issue.
2. Update the `README.md` with details of changes to the interface or endpoints.
3. Verify that your code follows standard Python PEP8 formatting.
4. Open your PR and we'll review it as soon as possible!

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.

Happy coding! 🚀
